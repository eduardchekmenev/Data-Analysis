# developmental/

A free development area for prototyping new scripts, testing modifications to the pipeline, and exploring new analysis approaches.

## Purpose

Use this folder to:
- Draft new pipeline steps or GUI tools before they are ready to be proposed for `Working Data Pipeline/`
- Test modifications to existing pipeline logic without touching locked scripts
- Explore alternative processing approaches (e.g., different denoising methods, fitting models)
- Stage work-in-progress code you want to share or collaborate on

## Guidelines

- No restrictions on what you commit here — this area is intentionally open
- Organize your work in a subfolder named after yourself or your feature (e.g., `developmental/your-name/` or `developmental/new-fitting-approach/`)
- Add a brief comment block or small README to your subfolder describing what the code does
- When a script matures and is ready for the main pipeline, open a discussion or PR to propose it for `Working Data Pipeline/`

## Status

Scripts here are **not considered stable or reviewed**. Do not rely on them for production data analysis.
